PLATAFORMA DE MICROSERVICIOS EN AWS EC2

Desarrollo de APIs REST y microservicios desplegados en EC2 con base de datos RDS

Objetivo del Laboratorio

Implementar una plataforma de microservicios en AWS utilizando EC2 para el despliegue de APIs REST con FastAPI y RDS como base de datos MySQL.

  • Configurar instancia EC2 con conectividad a base de datos RDS
  • Configurar Security Groups para permitir tráfico en puertos 8000-8004
  • Crear base de datos mediante cliente MySQL
  • Desarrollar script para ejecutar todos los microservicios
  • Verificar ejecución exitosa de microservicios

Arquitectura de la Solución

EC2 Instance

Tipo: t3.medium

Sistema: Ubuntu 20.04 LTS

Propósito: Host de microservicios

RDS Database

Tipo: MySQL 8.0

Instancia: db.t3.small

Propósito: Almacenamiento de datos

Security Groups

EC2: SSH + HTTP 8000-8004

RDS: MySQL 3306

Propósito: Control de acceso
