Laboratorio: Redundancia con HSRP, OSPF Multiárea y EtherChannel

Documentación completa del laboratorio CCNA - Conmutación y Enrutamiento Avanzado

1) Objetivo

Implementar una red empresarial con alta disponibilidad que integre:

  • OSPF Multiárea (Área 0 y Área 1) para enrutamiento dinámico.
  • HSRP para redundancia de gateway por VLAN con balanceo de carga.
  • EtherChannel (LACP) para agregación de enlaces entre switches.
  • Switch multicapa con Inter-VLAN routing.
  • Verificación de tablas de enrutamiento y estados de redundancia.

2) Topología y Roles

  • IOU1, IOU2, IOU5: Routers del backbone OSPF (Área 0).
  • SW1: Switch multicapa (Área 1) con SVIs para VLANs 10, 20, 30, 40.
  • IOU4: Switch multicapa secundario para redundancia HSRP.
  • HSRP: Configuración con balanceo de carga (VLANs 10-20 activas en SW1, VLANs 30-40 activas en IOU4).
  • EtherChannel: Enlace agregado entre IOU4 y otro dispositivo (no especificado).

4) Configuración

4.1 Configuración de Interfaces y Direccionamiento IP

IOU2 (Router):

interface Ethernet0/0
ip address 10.1.1.2 255.255.255.252
no shutdown
!
interface Ethernet0/1
ip address 172.16.1.1 255.255.255.252
no shutdown
!
interface Loopback1
ip address 2.2.2.2 255.255.255.255

SW1 (Switch Multicapa):

interface Ethernet1/0
no switchport
ip address 172.16.1.2 255.255.255.252
no shutdown
!
interface Loopback1
ip address 4.4.4.4 255.255.255.255

IOU5 (Router):

interface Ethernet0/1
ip address 10.1.2.2 255.255.255.252
no shutdown
!
interface Loopback1
ip address 3.3.3.3 255.255.255.255

4.2 Configuración de OSPF Multiárea

IOU2:

router ospf 1
router-id 2.2.2.2
network 2.2.2.2 0.0.0.0 area 0
network 10.1.1.0 0.0.0.3 area 0
network 172.16.1.0 0.0.0.3 area 1

SW1:

router ospf 1
router-id 4.4.4.4
network 4.4.4.4 0.0.0.0 area 1
network 172.16.1.0 0.0.0.3 area 1
network 192.168.10.0 0.0.0.255 area 1
network 192.168.20.0 0.0.0.255 area 1
network 192.168.30.0 0.0.0.255 area 1
network 192.168.40.0 0.0.0.255 area 1

IOU5:

router ospf 1
router-id 3.3.3.3
network 3.3.3.3 0.0.0.0 area 0
network 10.1.2.0 0.0.0.3 area 0

4.3 Configuración de HSRP para Redundancia

SW1 (Activo para VLANs 10 y 20):

interface Vlan10
ip address 192.168.10.1 255.255.255.0
standby 10 ip 192.168.10.254
standby 10 priority 110
standby 10 preempt
!
interface Vlan20

standby 20 ip 192.168.20.254
standby 20 priority 110
standby 20 preempt
!
interface Vlan30

standby 30 preempt

standby 40 preempt

IOU4 (Respaldo para VLANs 30 y 40):

!

4.4 Configuración de Trunking y EtherChannel (IOU4)

default interface range ethernet 0/0 - 1
interface range ethernet 0/0 - 1
switchport trunk encapsulation dot1q
switchport mode trunk
channel-group 1 mode active
