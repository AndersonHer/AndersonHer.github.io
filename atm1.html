Laboratorio: Automatización de Configuración de Dispositivos de Red

Documentación completa del laboratorio de automatización con Netmiko

1) Objetivo del Laboratorio

Automatizar la configuración básica de múltiples dispositivos de red (routers y switches Cisco) utilizando Netmiko, una biblioteca de Python para interactuar con dispositivos de red via SSH y Telnet.

2) Topología

La topología incluye los siguientes dispositivos:

  • R1: Router Cisco (conexión SSH)
  • SW1, SW2, SW3: Switches Cisco (conexión Telnet)

Todos los dispositivos están en la red 192.168.56.0/24.

Figura 1: Topología de red con dispositivos Cisco

3) Script de Automatización (provisionamiento.py)

Figura 2: Captura del script de automatización

Código completo del script:

from netmiko import ConnectHandler

# Lista de dispositivos
devices = [
    # Router con SSH
    {
        "device_type": "cisco_ios",
        "ip": "192.168.56.101",
        "username": "cisco",
        "password": "cisco",
        "display_name": "R1"
    },

    # Switches con Telnet
    {
        "device_type": "cisco_ios_telnet",
        "ip": "192.168.56.102",
        "port": 23,
        "username": "ciscoSW1",
        "password": "cisco",
        "secret": "cisco",
        "display_name": "SW1"
    },
    {
        "device_type": "cisco_ios_telnet",
        "ip": "192.168.56.103",
        "port": 23,
        "username": "ciscoSW2",
        "password": "cisco",
        "secret": "cisco",
        "display_name": "SW2"
    },
    {
        "device_type": "cisco_ios_telnet",
        "ip": "192.168.56.104",
        "port": 23,
        "username": "ciscoSW3",
        "password": "cisco",
        "secret": "cisco",
        "display_name": "SW3"
    }
]

# Configuraciones aplicadas
common_config = [
    "username admin privilege 15 secret cisco123",
    "enable secret cisco123",
    "banner motd ^C---No autorizado--- ingrese con credenciales validas---^C",
    "logging console"
]

# Manejo de errores
for device in devices:
    try:
        print(f"\nConectando a {device['display_name']} ({device['ip']})...")
        connection = ConnectHandler(**{k: v for k, v in device.items() if k != "display_name"})
        connection.enable()
        
        config_commands = [f"hostname {device['display_name']}"] + common_config
        connection.send_config_set(config_commands)
        
        output = connection.send_command("show running-config | include hostname")
        print(f"Configuración aplicada en {device['display_name']}: {output}")
        
        connection.disconnect()
        
    except Exception as e:
        print(f"Error con {device['display_name']}: {str(e)}")
