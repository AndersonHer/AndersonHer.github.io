Laboratorio: Telefonía IP, QoS y EIGRP

Sede principal con telefonía IP, calidad de servicio y enrutamiento dinámico

Objetivo del Laboratorio

Implementar una red escalable con los siguientes requisitos:

  • Telefonía IP en los departamentos de Ventas, Soporte y Administradores.
  • Calidad de Servicio (QoS) para priorización del tráfico de voz.
  • Conmutación VLAN para segmentación departamental.
  • EIGRP como protocolo de enrutamiento dinámico entre sedes.
  • Conectividad total entre la sede principal y las sedes remotas.

Configuración de Routers

R1 – Sede Principal

hostname R1
!
interface Serial0/3/0
 ip address 10.0.0.1 255.255.255.252
 clock rate 2000000
 no shutdown
!
interface FastEthernet0/0
 no ip address
 service-policy output QOS-DEKKO
 duplex auto
 speed auto
 no shutdown
!
interface FastEthernet0/0.10
 encapsulation dot1Q 10
 ip address 192.168.10.1 255.255.255.0
!
interface FastEthernet0/0.20
 encapsulation dot1Q 20
 ip address 192.168.20.1 255.255.255.0
!
interface FastEthernet0/0.30
 encapsulation dot1Q 30
 ip address 192.168.30.1 255.255.255.0
!
router eigrp 100
 network 10.0.0.0 0.0.0.3
 network 192.168.10.0
 network 192.168.20.0
 network 192.168.30.0
 no auto-summary
!
ip dhcp excluded-address 192.168.20.1
ip dhcp pool Vlan-voz
 network 192.168.20.0 255.255.255.0
 default-router 192.168.20.1
 option 150 ip 192.168.20.1

Configuración del Switch SWL3

Puerto Troncal

interface FastEthernet0/1
 switchport trunk encapsulation dot1q
 switchport mode trunk

Puertos de Acceso y Voz

interface FastEthernet0/2
 switchport access vlan 10
 switchport voice vlan 20

interface FastEthernet0/24
 switchport access vlan 10
 switchport voice vlan 20

Puerto de Administración

interface FastEthernet0/3
 switchport access vlan 30
